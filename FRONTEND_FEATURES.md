- **Frontend**: React 18 + TypeScript
- **Visualization**: Recharts
- **Styling**: Tailwind CSS
- **AI Integration**: Gemini 3 Flash API
- **State Management**: React Hooks + localStorage
- **Build**: Webpack/Vite
